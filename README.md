# Chomado

Chomado command

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'chomado', :git => "git://github.com/tbpgr/chomado.git"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install specific_install
    $ gem specific_install -l "git://github.com/tbpgr/chomado.git"

## Usage

### commands

~~~bash
$ chomad commands
homo
lolita
~~~

### homo

~~~bash
$ chomad homo
(^o^)ホモォ...
~~~

### lolita

~~~bash
$ chomad lolita
はるかぜﾀｿ ﾊｧﾊｧ
~~~
